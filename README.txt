Данный мод заменяет стандартную озвучку экипажа на пользовательскую, сделанную VLBRTN для себя и других.

Эта озвучка действует ТОЛЬКО на экипаж, то есть реплики, не принадлежащие экипажу (Мы захватили точку *, радиопереговоры, чат) не заменены
--------------------------------------------------------------------------
Установка

- Найти в корневой папке игры файл config.blk и открыть его любым текстовым редактором
- Найти блок sound и в нем добавить строчку enable_mod:b=yes
   
   Должно быть так:
   
sound{
  speakerMode:t="stereo"
  fmod_sound_enable:b=yes
  enable_mod:b=yes
}

- Распаковать папку mod в папку sound
